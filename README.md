# DotNetBank

## Описание
Това е учебно WinForms приложение, което демонстрира базови банкови операции като създаване на сметки, внасяне и теглене на средства и преглед на историята на транзакциите. Коментарите в кода са добавени от Petar Ivanov (F116389) и описват подробно всяка част от логиката.
